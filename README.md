# voting-app

## Project setup
```
npm install
```

## Project START
```
npm start => http://localhost:4000
```



### Development: Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

Docker:
docker-compose up --build
